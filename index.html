Checkout page

I had to rethink the whole logic behind storing items in the shopping cart. I start by creating a local-storage with the name "games", if one does not already exist. The value is stord as an array, and is parsed to a array named gamesInCart. When a user clicks "add to cart" button, it checks if the game.id alredy exist in the gamesInCart array, if it dose not it gets added to the array, it then takes the array and replace it with the old array stored in local storage. To get the amout of items stored in local storage with the key of games i JSON.parse the value to a variable and do a variable.lenght to get the amout of values in the array, then display the amount to the cart icon. By having each games.id stored in the gamesInCart array, i can loope throu the array in the checkout page, for each id in the array i do a fetch for the game with the spesific id.

Of course, the check out page is not quite finished it should be possible to select the number of each game and update the total price and be able to delete games from the shopping cart.
